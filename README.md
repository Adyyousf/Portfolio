## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/Adyyousf/portfolio

# Access
$ cd portfolio

# Run the project
$ python manage.py runserver

# The server will initialize in the <http://127.0.0.1:8000>
```
